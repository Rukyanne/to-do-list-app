# To-Do List Command Line Application (TODO-001)

A simple, persistent command-line interface (CLI) application built with Python to manage personal tasks. This project was developed following the core phases of the Software Development Life Cycle (SDLC).
## ðŸŒŸ Key Features

The application allows users to:

* **Add Tasks:** Quickly input new tasks to the list.
* **View Tasks:** Display all current tasks with their index and completion status.
* **Mark Tasks Complete:** Change a task's status from pending `[ ]` to complete `[X]`.
* **Delete Tasks:** Permanently remove tasks from the list.
* **Data Persistence:** Tasks are automatically saved to a local file (`tasks.txt`) between sessions.
